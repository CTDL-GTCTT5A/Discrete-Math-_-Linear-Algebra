This project solve some of simple problem relative to DiscreteMath and LinearAlgebra.


Karnaugh-Map:
  + Find all of big cell.

Vector:

  + Multiple vector with value k.
  
 
Calculate_Matrix:

  + Inverse Matrix Method.
  + Multiply matrix.
  + Rank of matrix.
  + Linear equations
